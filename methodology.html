Methodology

P 值是怎么算出来的

我们把"当前市场离泡沫顶部还有多远"压缩成一个 0 到 1 的数字。这一页完整公开计算方法，以及它诚实的局限——我们相信透明比神秘更可信。

1单信号顶部接近度

对每一个监测指标 i，我们把它当前的值，放在"常态"和"顶部"之间做线性定位，得到一个 0–1 的接近度：

// 单信号接近度 sᵢ(t) = ( xᵢ(t) − xᵢnormal ) / ( xᵢtop − xᵢnormal )
  • xᵢnormal　常态基准 = 1997–98 上半年的中位数
  • xᵢtop　顶部值 = 2000-03-10 那天的读数

结果含义：0 = 回到泡沫前的常态，1 = 已经和 2000 年顶部一样极端。

2综合压力 P

把所有信号的接近度按权重加总，得到综合泡沫压力 P：

// 综合压力 P(t) = Σ wᵢ · sᵢ(t)

0 = 常态，1.0 = 2000 年 3 月顶部状态。这就是首页那个大数字。

3月度 hazard（顶部风险率）

把压力 P 通过一个逻辑斯蒂函数，映射成"当月见顶的风险率"：

// 逻辑斯蒂映射 h(t) = 1 / ( 1 + e−[ k(P − P₀) + β·τ ] )

k、P₀、β 是形状参数，τ 是泡沫已持续的时间。

4相似度匹配（最可靠的输出）

在 1997–2000 的历史中，找出哪一天的"信号组合"和今天最接近——这是判断"我们在历史的哪个位置"最稳的方法：

// 找最相似的历史位置 位置 = argmind Σ wᵢ · | sᵢ(今天) − sᵢ(历史第 d 天) |
关键约束：匹配必须限定在上行阶段（过去 60 日动量 > 0），否则会把今天误匹配到"顶部之后的下行段"——那是致命的方向错误。

!诚实的边界（务必读这段）

这套公式结构是严谨的，但我们不假装它是水晶球：

  • 参数无法严格拟合：泡沫级别的顶部，历史上只有 2000 年这一个主样本，所以 k / P₀ / β 只能合理设定，不能用统计严格回归出来。
  • 顶部值可以精确标定：xᵢtop 用 2000 年的真实历史数据，这部分是扎实的。
  • 最可靠的两个输出是"P 随时间的变化轨迹"和"相似度匹配位置"，而不是某个月份的绝对见顶概率。
一句话总结：P 是一支动态温度计，不是水晶球。它告诉你"现在有多热、在朝哪个方向走多快"，而不是"哪一天会崩"。
